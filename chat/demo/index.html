🚀 Chat Widget Demo

Embeddable chat widget for any website

Features

✨ Modern UI Sleek floating widget with smooth animations
🔧 Easy Setup Just 2 lines of code to embed
🌐 CORS Ready Works on any domain
🔐 Secure API key authentication included
📱 Mobile Friendly Responsive design for all devices
⚡ Lightweight Single JS file, ~8KB gzipped

Quick Start

Step 1: Add Script Tag

<script src="http://localhost:3000/chat-widget.js"></script>

Step 2: Initialize Widget

ChatWidget.init({ apiUrl: 'http://localhost:3000', apiKey: 'test-key-123', title: 'Chat Support', placeholder: 'Type a message...' });
✓ The chat widget is already active on this page! Look for the purple chat bubble in the bottom-right corner.

Configuration Options

ChatWidget.init({ // Required apiUrl: 'https://your-server.com', apiKey: 'your-secret-key', // Optional title: 'Chat Support', // Widget title placeholder: 'Type a message...',// Input placeholder theme: 'light', // 'light' or 'dark' position: 'bottom-right', // 'bottom-right' or 'bottom-left' width: '400px', // Widget width height: '600px' // Widget height });

Deployment

Backend Setup

cd backend npm install npm start
Server runs on http://localhost:3000 by default

For Production

  • Host backend on cloud (Heroku, AWS, DigitalOcean, etc.)
  • Update API_KEY in .env to something secure
  • Update CORS origins if needed
  • Point widget script to your production URL
  • Connect to your AI service (OpenAI, etc.)

API Endpoints

POST /chat
Send a message and get reply
Body: { message: "Hello", sessionId: "session-123", timestamp: "2024-01-01T00:00:00Z" }
GET /session/:sessionId
Get conversation history
DELETE /session/:sessionId
Clear a conversation
