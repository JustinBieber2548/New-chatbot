📚 Integration Examples

1. Basic Setup

Simplest way to add the chat widget to your website.

<!-- Add before closing </body> --> <script src="http://localhost:3000/chat-widget.js"></script> <script> ChatWidget.init({ apiUrl: 'http://localhost:3000', apiKey: 'test-key-123' }); </script>
✓ This example is active on this page

2. Advanced Configuration

Customize the widget appearance and behavior.

ChatWidget.init({ apiUrl: 'https://api.yourcompany.com', apiKey: 'sk-prod-key-here', title: 'Customer Support', placeholder: 'Ask me anything...', theme: 'light', position: 'bottom-left', width: '380px', height: '550px' });

3. Event Listeners

Listen to chat events and react to them in your code.

// Custom event handling window.addEventListener('chat-widget-ready', function() { console.log('Widget initialized'); }); window.addEventListener('chat-widget-message', function(e) { console.log('New message:', e.detail); // Send to analytics gtag('event', 'chat_message', { session_id: ChatWidget.state.sessionId, message: e.detail.text }); });
Note: You'll need to implement custom event dispatching in chat-widget.js

4. Dynamic Positioning

Change widget position based on user preference.

// Bottom Right (Default) ChatWidget.init({ apiUrl: 'http://localhost:3000', position: 'bottom-right' }); // Bottom Left ChatWidget.init({ apiUrl: 'http://localhost:3000', position: 'bottom-left' });

5. Pre-filled Messages

Send initial messages programmatically.

// Initialize widget ChatWidget.init({ apiUrl: 'http://localhost:3000', apiKey: 'test-key-123' }); // Open widget and send message setTimeout(() => { ChatWidget.toggleWidget(); ChatWidget.sendMessage( 'Hi, I have a question about your pricing', ChatWidget.elements.messagesContainer, ChatWidget.elements.input ); }, 1000);

6. Multiple Widgets (Advanced)

Run multiple chat widgets for different services.

// Support Widget ChatWidget.init({ apiUrl: 'http://localhost:3000/chat', apiKey: 'support-key', title: 'Support', position: 'bottom-right' }); // Sales Widget (different instance) // You would need to create a separate class or modify // the widget to support multiple instances
This requires modifying ChatWidget to support multiple instances. Current version supports one widget per page.

7. Analytics Integration

Track chat interactions with Google Analytics or similar.

// Google Analytics function trackChatEvent(eventName, data = {}) { gtag('event', eventName, { session_id: ChatWidget.state.sessionId, ...data }); } // Track when widget opens const originalToggle = ChatWidget.toggleWidget; ChatWidget.toggleWidget = function() { originalToggle.call(this); trackChatEvent('chat_widget_opened', { is_open: this.state.isOpen }); };

Setup for production environment.

✓ Store API keys securely
✓ Implement rate limiting

9. Debugging & Troubleshooting

Check widget status and debug issues.

// Check if widget is loaded console.log('Widget available:', window.ChatWidget ? 'Yes' : 'No'); // Check widget state console.log('Session ID:', ChatWidget.state.sessionId); console.log('Is open:', ChatWidget.state.isOpen); console.log('Messages:', ChatWidget.state.messages); // Test API connection fetch('http://localhost:3000/health') .then(res => res.json()) .then(data => console.log('API Status:', data)) .catch(err => console.error('API Error:', err));
